# 极简日记本

##如何使用
暂支持生成 txt 文件的日记

终端输入:
```
python main.py 文件名a.txt
```

文件名a 是新建的日记名称, 

如果文件名已经存在, 


## 需求
* 一次接收输入一行日记
* 保存为本地文件
* 再次运行系统时, 能打印出过往的所有日记

## 需求分析
需求理解有偏差
###~~1.记录内容 ~~
* ~~输入内容~~
* ~~新建文档-打开文档-写入内容-关闭文档~~


###2.查询日记
* ~~输入查询文件名~~
* ~~打开文件-读内容-显示内容~~


###3.细节考虑
* 每步的文字提示
* 输入后的反馈
* 如果新建的文件名已经存在, 怎么办? 追加日记?
* 查询的文件名如果不存在, 提示什么?
* 其他细节

**思路:**先将1和2完成后, 在调整细节.
